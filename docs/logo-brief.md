## 1. 서비스 개요

**WeWantHappy**는 전 세계 분쟁·긴장 상황을 실시간으로 추적하고, 사용자에게 알림과 지도로 제공하는 세계정세 인텔리전스 플랫폼입니다.

> "지구 어딘가에서 지금 무슨 일이 일어나고 있는지, 한눈에."

### 핵심 기능
- 🗺️ **실시간 세계 지도** — 분쟁·이슈 클러스터 마커
- 📊 **긴장도 지수 (Tension Index)** — 국가/지역별 위험 수치 시각화
- 🔔 **즉각 알림** — 주요 사건 발생 시 푸시 알림
- 💬 **커뮤니티** — 사용자 간 세계정세 토론
- 📱 **PWA + 안드로이드** — 앱처럼 사용하는 웹 서비스

### 수익 모델
- Free: 기본 지도·알림
- Pro / Pro+: 이력 데이터, 고급 분석, 우선 알림

## 2. 브랜드 정체성

### 이름의 의미
**We Want Peace** — "우리는 평화를 원한다"

- 전쟁·분쟁을 다루지만, 목적은 **평화를 향한 인식**
- 사용자는 공포가 아닌 **정보로 세상을 이해**한다
- 집단적 바람("We") — 개인이 아닌 세계 공동체의 목소리

### 브랜드 톤
| 속성 | 설명 |
|------|------|
| 진지함 | 가볍지 않다. 실제 세계 이슈를 다룬다 |
| 신뢰성 | 데이터 기반, 편향 없는 정보 |
| 접근성 | 누구나 이해할 수 있게 시각화 |
| 희망 | 분쟁을 보여주되, 목적은 평화 |

### 브랜드 포지셔닝
- Bloomberg / Reuters 급의 **신뢰감**
- Duolingo / Notion 급의 **친근한 접근성**
- 너무 무겁지 않게, 너무 가볍지 않게

## 3. 로고 방향성

### 핵심 컨셉 (3가지 중 선택 또는 혼합)

**A. 지구 + 시선**
- 지구본/지도 위에 눈 또는 레이더 느낌
- "세상을 주시한다"는 의미
- 단순한 원형 지구 실루엣에 마커 핀 하나

**B. 평화 기호의 재해석**
- 기존 ☮ 평화 기호를 현대적으로 재해석
- 지도 핀, 위성 신호, 데이터 포인트 결합
- 너무 클리셰가 되지 않도록 추상화 권장

**C. 신호 + 파동**
- 무선 신호(WiFi·레이더) 형태 → 정보가 퍼져나가는 이미지
- 중심점에서 파동이 번지는 형태
- 긴장도 지수의 "파동" 시각화와 연결

## 4. 디자인 스펙

### 색상 팔레트 (권장)

| 역할 | 색상 | HEX | 비고 |
|------|------|-----|------|
| 메인 | 딥 네이비 | `#0F172A` | 밤하늘, 세계지도 배경 |
| 포인트 | 에메랄드 그린 | `#10B981` | 평화, 희망, 안전 |
| 강조 | 앰버 | `#F59E0B` | 경고, 긴장도 높음 |
| 위험 | 로즈 레드 | `#EF4444` | 분쟁, 위험 지역 |
| 보조 | 스카이 블루 | `#38BDF8` | 정보, 신뢰 |

> **핵심:** 메인 컬러는 딥 네이비 + 에메랄드 조합 권장
> 어두운 배경(앱 다크모드)에서도 선명하게 보여야 함

### 폰트 방향
- **서체 스타일:** 지오메트릭 산세리프 (Geist, Inter, Plus Jakarta Sans 계열)
- **느낌:** 현대적, 기술적, 읽기 쉬움
- **대소문자:** `WeWantHappy` (카멜케이스) 또는 `WEWANTPEACE` (올캡스) 검토
- **한국어 표기:** 위원트피스 (병기 불필요, 영문 단독 사용)

### 형태 요구사항
- 심볼 + 워드마크 조합형 (기본)
- 심볼 단독 사용 가능한 형태 (앱 아이콘, 파비콘용)
- 1:1 정사각형 버전 필수 (앱 아이콘)
- 가로형 버전 필수 (헤더, 문서용)

## 9. 기타 참고사항

- 서비스 특성상 **"무거운 뉴스를 가볍게 이해한다"**는 역설이 핵심
- 전쟁 이미지를 직접 연상시키는 요소는 지양
- 글로벌 서비스 → 특정 국가/문화 편향 없는 보편적 디자인
- 앱 심볼은 **지도 위의 점(마커) 하나** 수준으로 단순하게도 가능
